Third-party tools
#################

Thanks to the contributors, the following tools and integrations are available:

Import tools and integration
~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- `Amazfish <https://amazfish.github.io/>`_ (companion application for smartwatch, integration with FitTrackee from v2.9.0)
- `FitTrackee Uploader <https://github.com/ebrithiljonas/fittrackee-uploader>`_ (companion application for FitTrackee to upload workout files)
- `garmin-to-fittrackee <https://git.dryusdan.fr/Dryusdan/garmin-to-fittrackee>`_ (script to synchronize garmin to fittrackee)
- `strava-to-fittrackee <https://github.com/jat255/strava-to-fittrackee>`_ (connector to pull workouts from a Strava account and upload them to a FitTrackee instance)

Installation scripts
~~~~~~~~~~~~~~~~~~~~
- `fittrackee_installer.sh <https://github.com/speefak/fittrackee_installer.sh>`_ (install fittrackee on debian 12 netinstall)
