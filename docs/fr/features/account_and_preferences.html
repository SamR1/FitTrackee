Compte et préférences¶

Compte¶

Modifié dans la version 0.3.0: réinitialisation du mot de passe

Modifié dans la version 0.6.0: changement d'adresse électronique

Modifié dans la version 0.7.13: export de données

Modifié dans la version 0.10.0: notification quand l'export de données est terminé

Les utilisateurs peuvent créer, modifier et supprimer leur compte.

Ils peuvent réinitialiser son mot de passe, modifier leur adresse e-mail et mettre à jour leur image de profil.

Ils peuvent également demander un export de leurs données.
Une archive zip sera générée, contenant 2 fichiers json (informations de l'utilisateur et données des séances) et tous les fichiers ajoutés.
Une notification est affichée une fois l'export terminé.

Note

Pour l'instant, il n'est pas possible d'importer ces fichiers sur une autre instance de FitTrackee.

Préférences¶

Modifié dans la version 0.5.0: ajout d'un thème d'interface et du système impérial

Modifié dans la version 0.6.11: préférence pour le dénivelé positif ajoutée

Modifié dans la version 0.7.3: format des date/heure ajouté

Modifié dans la version 0.7.15: ajout de l'affichage du graphe d'altitude

Modifié dans la version 0.7.16: ajout d'une stratégie de calcul de la vitesse maximale

Modifié dans la version 0.7.27: ajout du thème de l'interface

Modifié dans la version 0.9.0: Ajout de l'approbation des demandes d'abonnement, visibilité du profil dans le répertoire des utilisateurs, visibilité des séances

Modifié dans la version 0.10.0: ajout de la visibilité de la fréquence cardiaque

Modifié dans la version 0.10.3: évènement pour la création des segments

Modifié dans la version 0.11.0: affichage préférée pour les graphiques

Modifié dans la version 1.1.0: traitement des altitudes manquantes

Il est possible de définir les préférences d'interface :

  • langue, fuseau horaire, format de date et premier jour de la semaine

  • thème de l'interface (clair, sombre ou selon les préférences du navigateur)

Pour le compte, il est possible de définir :

  • approbation des demandes d'abonnement : manuellement (par défaut) ou automatiquement

  • visibilité du profil dans le répertoire des utilisateurs : masqué (par défaut) ou affiché

Pour les séances, il est possible de choisir :

  • entre le système métrique et le système impérial pour l'affichage de la distance, de l'altitude et de la vitesse

  • d'afficher ou de cacher les records et le total de dénivelé positif sur le tableau de bord

  • l'affichage préféré pour le graphique dans le détail de la séance :

    • toutes les données sur un seul graphique

    • chaque donnée affichée sur un graphique différent

  • le point de départ de l'axe du graphique d'altitude : altitude zéro ou minimale

  • d'exclure les valeurs extrêmes (qui peuvent être des erreurs GPS) lors du calcul de la vitesse maximale (par défaut, les valeurs extrêmes sont exclues)

    Note

    La modification de cette préférence n'affectera que les prochains fichiers ajoutés.

  • le traitement des altitudes manquantes si un service de données d'altitude est défini :

    • aucun

    • OpenElevation (données brutes)

    • OpenElevation (données lissées)

    • Valhalla

  • la visibilité par défaut des séances, de l'analyse, de la carte et de l'affichage de la fréquence cardiaque

Le type de pause pouvant générer un segment lors de l'ajout d'un fichier .fit peut être défini :

  • tous les évènements de type pause,

  • seulement les pauses manuelles,

  • aucun.

Préférences des sports¶

Ajouté dans la version 0.5.0.

Modifié dans la version 0.8.0: ajout de l'équipement par défaut

Modifié dans la version 1.1.0: ajout de l'affichage Allure/Vitesse

Les préférences de sport suivantes peuvent être définies :

  • changer la couleur du sport (utilisée pour l'image du sport et les graphiques)

  • modifier le seuil de vitesse d'arrêt (pour les prochains fichiers gpx ajoutés)

  • désactiver/activer un sport.

  • définir des équipements par défaut.

  • configurer de l'affichage de la vitesse/allure pour la randonnée, la course, le trail et la marche :

    • allure

    • vitesse

    • allure et vitesse moyenne

    Note

    Si un sport est désactivé par un administrateur, il ne peut pas être activé par un utilisateur. Dans ce cas, il n'apparaîtra dans les préférences que si l'utilisateur a des séances et seule la couleur du sport peut être modifiée.
    Un sport désactivé (par l'administrateur ou l'utilisateur) n'apparaîtra pas dans la liste lors de l'ajout d'une séance.
    Une séance avec un sport désactivé sera toujours affichée dans l'application.

Une fonction d'édition rapide est également disponible pour modifier la couleur, activer le sport, mettre à jour le seuil de vitesse d'arrêt ou réinitialiser les préférences.

Demandes de suivi¶

Ajouté dans la version 0.9.0.

Les utilisateurs peuvent afficher les demandes d'abonnements pour les accepter ou les rejeter.

Ils peuvent afficher les utilisateurs bloqués.

Modération¶

Les utilisateurs peuvent afficher les avertissements et sanctions reçues et faire appel.

Notifications¶

Ajouté dans la version 0.9.0.

Les utilisateurs peuvent définir les préférences des notifications.

Messages¶

Ajouté dans la version 1.0.0.

Les utilisateurs peuvent définir les préférences des messages.

Chargement d'archives¶

Ajouté dans la version 0.10.0.

Les utilisateurs peuvent voir, interrompre et supprimer les tâches de chargement d'archives.
