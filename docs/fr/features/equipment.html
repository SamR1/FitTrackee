Équipements¶

Ajouté dans la version 0.8.0.

Modifié dans la version 0.9.1: ajout de la visibilité de l'équipement

Modifié dans la version 0.9.3: ajout du Kayak et du Canoë au type Bateau

Modifié dans la version 0.9.7: sports mis à jour pour le type Vélo

Modifié dans la version 0.9.10: ajout du type Planche

Modifié dans la version 0.10.3: ajout du Stand up paddle au type Planche

Modifié dans la version 0.11.0: ajout du Tennis (Extérieur) au type Chaussures

Modifié dans la version 1.0.0: ajout du Padel (Extérieur) au type Chaussures

Modifié dans la version 1.1.0: ajout de Canöe en eaux vives et du Kayak en eaux vives au type Bateau

Les utilisateurs peut créer des équipements qui peuvent être associés à des séances.

Les types d'équipement suivants sont disponibles, selon le sport :

  • Vélo : Halfbike, Vélo (Sport, Transport, Trekking), VTT et VTT (Électrique)

  • Vélo d'entraînement : Vélo (Virtuel),

  • Planche: Planche à voile et Stand up paddle

  • Kayak/Bateau : Aviron, Canöe, Canöe en eaux vives, Kayak et Kayak en eaux vives

  • Chaussures : Alpinisme, Course, Marche, Padel (Extérieur), Randonnées, Tennis (Extérieur) et Trail

  • Skis : Ski (Alpin et Randonnée)

  • Raquettes : Raquettes.

Les utilisateurs peuvent définir la visibilité de l'équipement :

  • privé: seul le propriétaire peut voir l'équipement dans le détail de la séance,

  • abonnés seulement: seul le propriétaire et les abonnés peuvent voir l'équipement dans le détail de la séance,

  • public: tout le monde peut voir l'équipement dans le détail de la séance, même les utilisateurs non authentifiés.

Le détail de l'équipement n'est visible que par son propriétaire.

Pour le moment, un seul équipement peut être associé à une séance.

Les totaux suivants sont affichés pour chaque équipement :

  • distance totale

  • durée totale

  • nombre total de séances

Note

En cas de total incorrect (bien que cela ne doive pas se produire), il est possible de recalculer les totaux.

Il est possible de définir un équipement par défaut pour les sports : à l'ajout de la séance, l'équipement sera automatiquement affiché dans la liste déroulante selon le sport sélectionné.

Un équipement peut être modifié (label, type d'équipement, description, visibilité, status actif et sports par défaut).

Avertissement

Le changement de type d'un équipement supprimera toutes les associations existantes à des séances pour cet équipement et les sports définis par défaut.

Un équipement désactivé n'apparait plus dans la liste déroulante lors de l'ajout d'une séance. Il reste affiché dans le détail de la séance, à laquelle il a été associé avant sa désactivation.

Note

Un type d'équipement peut être désactivé par un administrateur.
