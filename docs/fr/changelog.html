Version 1.0.0 (05/10/2025)¶

FitTrackee permet désormais de filtrer les entraînements par lieu et de les afficher sur une carte.
Pour ce faire, FitTrackee nécessite l'installation de l'extension PostGIS sur la base de données PostgreSQL et le recalcul de toutes les séances afin de générer des données géospatiales.
Vous trouverez les instructions pour la mise à niveau dans la documentation).
Deux nouvelles variables d'environnement ont été ajoutées : ENABLE_GEOSPATIAL_FEATURES et NOMINATIM_URL.

Parmi les autres changements et améliorations, cette version apporte également un nouveau sport (le Padel en extérieur).

Attention :

  • La version 1.1.0 nécessitera que toutes les séances soient mises à jour.

  • Python 3.9 and PostgreSQL 12 ne sont plus supportés. Cette version ne prendra pas encore en charge Python 3.14 à sa sortie.

  • Cette version contient des migrations de base de données (voir les instructions dans la documentation).

Note: Cette note de version comprend les changements des versions bêta.
